Project Overview

CRESBOT is an AI-powered chatbot designed to streamline university-related queries for B.S. Abdur Rahman University. Built with like Hugging Face AI, CresBot provides real-time responses to FAQs about admissions, courses, fees, placements, and other academic details. With a sleek web-based interface, animations, and dark mode, CresBot ensures a seamless user experience.
